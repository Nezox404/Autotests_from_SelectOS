В данной директории представлен файл bash скрипта для проверки совместимости SelectOS и PostgreSQL, а так же вопросы к заказчику на тестовое задание и чек-лист. 
Запустить проект можно из под терминала. Использовались такие инструменты, как консоль, PostgreSQL, SelectOS. 

Реализованны проверки:
1. Создание БД и наполнения ее тестовыми данными
2. Имитация работы приложения (простое подключение и выборка)
3. Резервное копирование
4. Восстановление в новую БД
5. Очистка
